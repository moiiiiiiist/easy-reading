🚀 懒滚动 (Lazy Scrolling) 演示

💡 什么是懒滚动？

懒滚动是一种智能滚动策略，只在内容即将离开视口时才触发滚动，避免频繁的页面跳跃。

类似于翻页的体验：当前页面的内容快看完了，才翻到下一页。

🔄 传统滚动

  • 每次切换句子都滚动
  • 频繁的页面跳跃
  • 用户阅读被打断
  • 视觉干扰较大

🎯 懒滚动

  • 只在必要时滚动
  • 类似翻页体验
  • 减少视觉干扰
  • 更流畅的阅读体验

📱 可视化演示

第1句：在视口顶部的句子
第2句：正常可见的句子
第3句：正常可见的句子
第4句：当前正在阅读的句子
第5句：正常可见的句子
第6句：接近底部的句子
第7句：触发懒滚动！距离底部 < 150px

💡 当句子接近红色阈值线时，触发懒滚动，将下一句滚动到视口顶部

🔧 实现原理

1. 阈值检测

needsLazyScroll(element) { const rect = element.getBoundingClientRect(); const viewHeight = window.innerHeight; // 句子不在视口中？ if (rect.bottom <= 0 || rect.top >= viewHeight) { return true; } // 句子接近底部？（距离底部 < 150px） if (rect.bottom > (viewHeight - 150)) { return true; } return false; // 在安全区域，不需要滚动 }

2. 滚动触发

updateCurrentSentence() { const currentElement = getCurrentSentenceElement(); if (Utils.needsLazyScroll(currentElement)) { // 只在必要时滚动 Utils.smoothScrollToSentence(currentElement); } // 否则不滚动，保持当前视图 }

🎨 前端术语

相关的前端概念：

  • Lazy Scrolling - 懒滚动/延迟滚动
  • Viewport-based Scrolling - 基于视口的滚动
  • Pagination Scrolling - 分页式滚动
  • Smart Scrolling - 智能滚动
  • Threshold-based Scrolling - 基于阈值的滚动
  • Debounced Scrolling - 防抖滚动

🎯 使用场景

适合懒滚动

  • 文章阅读器
  • 电子书应用
  • 新闻阅读
  • 长文档浏览

不适合懒滚动

  • 精确定位需求
  • 短内容列表
  • 编辑器中的跳转
  • 搜索结果定位
